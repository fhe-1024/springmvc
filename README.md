自己总结的eclipse git 上传项目及下拉项目步骤
上传
1.已有的项目 右键team-shareproject-user or create repository in parent folder in project.
2.展开git repositories视图 在remote那里create remote 配置github url 必要时需要需要advanced选项
3.保存完后可以正常上传下载项目啦

下载 
1.展开git repository试图  clone a git repository and add the clone to this view
2.从正常的java ee项目import一个对应类型的项目即可(maven)
